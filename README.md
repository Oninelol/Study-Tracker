# Study Time Tracker

A simple Python project that tracks how much time I spend studying for each course.  
The goal of this project is to help students understand their study habits, compare workload across classes, and improve time management.

## Features

- Add study time for different courses
- Track total study hours per course
- View a summary of all recorded study sessions
- Compare how much time is spent on each class
- Simple and beginner-friendly Python implementation

## Example

Study Time Summary: 

- ECE 120: 5.5 hours
- MATH 241: 3.0 hours
- PHYS 212: 4.25 hours
- Total study time: 12.75 hours


## Motivation

As a student, it can be difficult to know where my time is going.
This project helps me organize my study schedule by recording the amount of time spent on each course.
By tracking study time, I can see which courses require the most effort. Helping me Plan my weekly schedule more effectively, balance time between technical and non-technical classes, and build better study habits over time.
